ENGINE ARCHITECTURE

Space Race is a 2.5D arcade shooter built using C++ and the Gateware middleware. Unlike engines with pre-built editors, this project required manual management of the game loop and memory.

I architected the core Main application loop and the Player class, handling the lifecycle of game entities from initialization to destruction. This included implementing a State Machine to handle transitions between the Main Menu, Gameplay, and Game Over states seamlessly.

AUDIO & MECHANICS

As the Audio Engineer, I designed and implemented a custom soundscape to fit the "Retro Arcade" aesthetic. I managed audio buffers to ensure sound effects triggered instantly upon collision events without stalling the main thread.

I also engineered the Power-Up System, featuring a "Nuke" ability. This required writing efficient iterators to traverse the active enemy list, applying damage or destruction states to all entities on screen simultaneously while maintaining performance.

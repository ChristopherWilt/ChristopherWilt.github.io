SECURITY ANALYSIS

Security in distributed systems begins with data visibility. For this project, I architected a User Session & Security Analysis System designed to parse and monitor raw web server logs.

I designed a normalized relational schema to track User IPs, Session Durations, and Access Patterns. Using complex SQL scripts and Stored Procedures, the system automatically flags anomalies—such as potential bot traffic or brute-force attempts—by analyzing User-Agent strings and request frequency.

QUERY OPTIMIZATION

Scale requires efficiency. I took a legacy dataset of 30,000+ vehicle records and re-architected it into Third Normal Form (3NF), reducing data redundancy by 60%.

To optimize read performance for the application layer, I analyzed query execution plans and implemented Clustered and Non-Clustered Indexes. This significantly reduced the execution time for complex joins, ensuring that the backend could handle high-volume search requests with minimal latency.
